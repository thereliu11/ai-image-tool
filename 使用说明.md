# AI教辅小红书自动作图工具 - 使用说明

## 📋 安装步骤

### 1. 安装Node.js
访问 https://nodejs.org 下载并安装 Node.js（推荐LTS版本）

### 2. 安装依赖
打开命令行，进入项目目录：
```bash
cd C:\Users\Administrator\Desktop\AI教辅作图工具
npm install
```

### 3. 启动应用
```bash
npm run dev
```

---

## 🎯 使用流程

### 第一步：配置API

1. 点击顶部"API设置"按钮
2. 输入你的 OpenAI API Key
3. 如需代理，填写代理主机和端口
4. 点击"保存设置"

### 第二步：导入商品

1. 准备好包含教辅图片的文件夹
2. 点击"导入商品"按钮
3. 选择图片文件夹（支持JPG/PNG格式）
4. 图片会按文件名排序显示在左侧列表

### 第三步：设置提示词

1. 在左侧"提示词设置"区域
2. 选择预设模板或自定义输入
3. 选择图片尺寸（推荐3:4竖版）

### 第四步：生成图片

1. 确认商品列表中的图片
2. 点击"批量生图"按钮
3. 等待生成完成（单张约1.5-2.5分钟）
4. 右侧可预览对比原图和生成结果

### 第五步：导出图片

1. 点击"AI教辅图导出"
2. 选择导出目录
3. 图片会自动保存到以文件夹名命名的子目录

---

## 🔧 功能说明

### 原创性增强（可选）

在"API设置"→"原创性增强"中开启：
- **叠加图片**：准备一些透明PNG图片作为叠加层
- **空间形变**：轻微调整图片尺寸
- **色彩偏移**：微调亮度、饱和度、色相

### 网络诊断

点击"网络诊断"按钮可测试API连接是否正常

---

## ⚠️ 注意事项

1. **API费用**：GPT Image 2按使用量计费，建议先用low质量测试
2. **网络环境**：国内用户可能需要配置代理
3. **图片大小**：单张图片建议小于20MB
4. **生成时间**：批量生成需要耐心等待

## 🐛 常见问题

**Q: 提示"API连接失败"怎么办？**
A: 检查API Key是否正确，网络是否通畅，如需代理请配置

**Q: 生成的图片文字不准确？**
A: 这是AI生成的正常现象，建议多次生成选择最佳结果

**Q: 如何提高生成质量？**
A: 使用high质量设置，增加提示词详细描述

**Q: 支持哪些图片格式？**
A: 支持JPG、JPEG、PNG格式
